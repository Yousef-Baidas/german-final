# Redemittel — written exam only (Momente B1.1, Lektion 1–10)

The exam is **written**, so this set **excludes oral-only phrases** (presentation delivery from
L4; *um das Wort bitten / sich nicht unterbrechen lassen* from L8). Grouped by writing function.

★ The single most exam-likely task is the **(halb-)formelle E-Mail (L9)** — lead with it.

## ① E-Mail schreiben (L9)

| German | Meaning | Use when |
|---|---|---|
| Sehr geehrte Frau …, / Sehr geehrter Herr …, | Dear Mrs/Mr … | Formal (geehrt**e** fem / geehrt**er** masc) |
| Sehr geehrte Damen und Herren, | Dear Sir or Madam | Name unknown |
| Liebe/r …, | Dear … | Semi-formal / known person |
| Ich schreibe Ihnen, weil … | I'm writing because … | Opening (weil → verb to end) |
| Vielen Dank für Ihre E-Mail. | Thanks for your email | Replying |
| Leider … | Unfortunately … | Softening bad news |
| Es tut mir leid, dass … | I'm sorry that … | Apologizing |
| Der Grund dafür ist, dass … | The reason is that … | Justifying |
| Wäre es möglich, …? / Wie wäre es mit …? | Would it be possible…? / How about…? | Proposing an alternative |
| Vielen Dank im Voraus. | Thanks in advance | You asked for something |
| Mit freundlichen Grüßen / Viele Grüße | Kind regards / Best wishes | Sign-off: formal / semi-formal |

## ② Meinung / argumentieren (L8)
- *Meiner Meinung nach …* — In my opinion… (verb comes right after: *…nach **ist** das gut.*)
- *Ich bin der Meinung, dass …* — I'm of the opinion that… (dass → verb to end)
- **zustimmen:** *Das sehe ich auch so. / Da stimme ich zu.*
- **widersprechen:** *Das sehe ich anders. / Ich bin anderer Meinung.*

## ③ Zweifel / Zuversicht (L2)
- *Versteh mich nicht falsch, aber …* — doubt (semi-formal tone)
- *Ich bin optimistisch, dass …* — confidence

## ④ Über Erlebnisse / Erfahrungen berichten (L1, L7)
- *Ich habe viele Erfahrungen mit … gemacht.*
- *Ich erinnere mich (daran), dass …*

## ⑤ Etwas beurteilen / bewerten (L5, L6)
- *Ein Vorteil/Nachteil ist, dass …*
- *Einerseits …, andererseits …*

## ⑥ Über eine Statistik (L10)
- *Die Statistik zeigt, dass …* · *… Prozent der …* · *Auffällig ist, dass …*
